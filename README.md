# 🌿 Grains & Garden — Premium Restaurant Web Concept

A highly interactive, design-forward, single-page responsive prototype engineered for **Grains & Garden**, an aesthetics-driven global-mix cafe and garden dining experience located in Civil Lines, Jhansi. 

This project was built from scratch to translate the restaurant's vibrant, curated Instagram visual identity into a high-performance, conversion-optimized digital storefront.

👉 **[Live Demo Link](https://eiracodes.github.io/grains-and-garden/)**

---

## ✨ Design Philosophy & Architecture

The core objective of this prototype was to deliver an incredibly premium UI/UX while adhering to specific architectural constraints for future production scalability.

*   **Instagram-First Aesthetic:** Implements a rich, earthy color palette—Deep Garden Green (`#1B3B2B`), Soft Oatmeal/Cream (`#F9F6F0`), and Ochre Terracotta accents—paired with high-contrast luxury serif typography (`Playfair Display`) and crisp sans-serif body copy (`Inter`).
*   **Elementor-Optimized Framework:** Built intentionally using a strict layout philosophy that mirrors WordPress + Elementor Pro container structures (Flexbox grids, explicit inner columns, and reusable global padding blocks). This ensures seamless, 1:1 visual fidelity when migrating to a production CMS.
*   **Zero Framework Overhead:** Written in pure, vanilla HTML5, CSS3 transitions, and clean JavaScript to ensure lightning-fast page speeds, flawless SEO semantic structure, and instantaneous mobile responsiveness.

---

## 🚀 Features Implemented

*   **1-Year Milestone Banner:** A beautifully integrated announcement bar celebrating the restaurant's first year anchoring the local cafe scene in Jhansi.
*   **Dynamic Interactive Menu:** A vanilla JS-powered tabbed filtering system (All, Neapolitan Pizza, Global Mix, Desserts & Drinks) featuring realistic INR pricing, diet indicators (Veg/Non-Veg), and elegant hover-zoom effects.
*   **Curated Visual Grid:** A stylized 3x2 gallery block mimicking an active Instagram feed, featuring smooth overlay metrics (Likes/Comments) to bridge social proof with the web experience.
*   **Conversion-Driven UX:** Dual-column reservation sections with native form validation focus-states, interactive sticky navigation headers, and a smooth-operating mobile hamburger drawer.

---

## 🛠️ Tech Stack & Specs

*   **Frontend:** HTML5, CSS3 (Flexbox/Grid, Custom Transitions)
*   **Scripting:** Vanilla JavaScript (ES6+)
*   **Typography:** Google Fonts (Playfair Display, Inter)
*   **Assets:** Optimized, responsive placeholder imagery via Unsplash API
